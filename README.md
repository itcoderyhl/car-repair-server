## 【原创】springboot+vue前后端分离的车辆维修管理系统

## 系统介绍

基于SpringBoot+Vue前后端分离的车辆维修管理系统系统，系统包含管理员、维修员、客户会员角色，功能如下：
- 管理员：首页统计（消费金额统计、会员统计）；会员管理；员工管理；车辆管理；维修管理（维修预约、维修工单、评价管理、零件管理）；礼品管理（礼品信息、礼品兑换）；充值管理；积分变更；留言管理；系统公告；系统设置（车辆品牌、会员等级、系统设置）；个人信息；密码修改。
- 维修工：维修管理（维修工单、评价管理、零件查看）；公告信息；修改个人信息；修改密码。
- 用户：注册、登录系统；车辆管理；维修管理（维修预约、维修工单、评价管理）；礼品管理（礼品信息、礼品兑换）；充值管理；积分变更；留言管理；公告信息；个人信息（充值）；密码修改。
系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，有需要可提供指导。

项目由本人原创开发，经过多次测试，项目运行无问题，请放心使用，作者QQ：1344127185。

## 使用技术

服务器端技术栈：

- Springboot
- SpringMvc
- Mybatis
- MySQL

前端技术栈：

- vue
- elementUI
- axios
- echarts

## 环境介绍

- IDEA集成环境
- JDK1.8
- MySQL5.6+

可提供所有运行软件、远程安装、技术指导。
